# DropDown

This is a general purpose collapsible dropdown (use it for dropdown list box (select), navigation drop down list...)



## Default props

<Playground>
  <ThemeProvider>
          <DropDown />
  </ThemeProvider>
</Playground>


## Props

<ThemeProvider>
  <Props of={DropDown} />
</ThemeProvider>


## Basic usage

<Playground>
  <ThemeProvider>
    <DropDown
      box={<div>Box content</div>}
      trigger={<div>trigger content</div>}
    />
  </ThemeProvider>
</Playground>


## Inline

<Playground>
  <ThemeProvider>
    <div>
      <p>Lorem ipsum</p>
      <DropDown
        inline={true}
        trigger={<div>Click me</div>}
        box={'Foo'}
      />
      <p>Lorem ipsum</p>
    </div>
  </ThemeProvider>
</Playground>
